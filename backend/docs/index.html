Events API Documentation

A RESTful API for managing events with full CRUD operations, built with FastAPI and deployed on AWS Lambda.

Base URL

https://YOUR-API-GATEWAY-URL/prod

Event Model

All events contain the following properties:

Property Type Required Description
eventId string Auto-generated Unique identifier for the event
title string ✓ Event title (1-200 characters)
description string ✓ Event description (1-2000 characters)
date string ✓ Event date in ISO format (YYYY-MM-DD)
location string ✓ Event location (1-500 characters)
capacity integer ✓ Maximum number of attendees (must be > 0)
organizer string ✓ Event organizer name (1-200 characters)
status string ✓ Event status: active, scheduled, ongoing, completed, cancelled

API Endpoints

Description: List all events with optional status filtering

Query Parameters:

Error Handling

The API uses standard HTTP status codes:

  • 200 OK: Request successful
  • 201 Created: Resource created successfully
  • 400 Bad Request: Invalid input or validation error
  • 404 Not Found: Resource not found
  • 500 Internal Server Error: Server error

CORS

CORS is enabled for all origins with the following configuration:

  • Allowed Origins: * (all)
  • Allowed Methods: GET, POST, PUT, DELETE, OPTIONS
  • Allowed Headers: * (all)
  • Credentials: Supported

Example Usage

cURL Examples

Create an Event

curl -X POST https://YOUR-API-URL/prod/events \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Tech Conference 2024",
    "description": "Annual technology conference",
    "date": "2024-12-15",
    "location": "San Francisco, CA",
    "capacity": 500,
    "organizer": "Tech Corp",
    "status": "active"
  }'

List All Events

curl https://YOUR-API-URL/prod/events

Filter Events by Status

curl https://YOUR-API-URL/prod/events?status=active

Get Specific Event

curl https://YOUR-API-URL/prod/events/EVENT-ID

Update Event

curl -X PUT https://YOUR-API-URL/prod/events/EVENT-ID \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Updated Event Title",
    "capacity": 600
  }'

Delete Event

curl -X DELETE https://YOUR-API-URL/prod/events/EVENT-ID

Architecture

  • API Gateway: REST API with CORS support
  • Lambda: Python 3.11 runtime with FastAPI and Mangum
  • DynamoDB: NoSQL database with pay-per-request billing
  • CDK: Infrastructure as Code for deployment
